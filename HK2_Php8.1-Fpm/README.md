# Docker Image - HK2_Php8.1-Fpm

## Description

This Image contains all extensions necessary for working with Magento

## Extension Added

> Some of the extension added in this image are

- bcmath
- calendar
- exif 
- gd
- intl
- mysqli
- opcache
- pdo_mysql
- soap
- sockets
- xsl
- zip

## Required Arguments 
- USER
- GIT_USER_NAME
- GIT_USER_EMAIL_ID
- EMAIL_NO_REPLY
